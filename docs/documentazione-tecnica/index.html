Documentazione Tecnica

Questa è la Documentazione tecnica completa cui fare riferimento per l’utilizzo dei componenti.
Si tratta della documentazione originale di Bootstrap-italia ricompilata per Regione Lombardia.

Leggete prima le Linee Guida

Per rispettare le Linee Guida per il design degli applicativi Web di Regione Lombardia e il Sistema di Identità Visiva Digitale di Regione Lombardia, va utilizzato solo un sottoinsieme sia delle varianti cromatiche e tipografiche, che dei componenti, presenti nella Documentazione tecnica, e che si hanno a disposizione in quanto ereditati dalla libreria originaria di Bootstrap Italia.

Le Linee Guida indicano quali componenti utilizzare, nella costruzione delle pagine di Web per le applicazione Regione Lombardia, e mostrano una serie di Esempi che possono essere utilizzati come base da cui partire.
