Che cos’è Bootstrap Regione Lombardia

Bootstrap Regione Lombardia è una personalizzazione di Bootstrap Italia per adeguarlo alle Linee Guida per il design degli applicativi Web di Regione Lombardia.

E’ quindi un fork del progetto Bootstrap Italia, presente su GitHub, da cui eredita i componenti, le funzionalità e tutti i relativi aggiornamenti e correzioni.

A sua volta Bootstrap Italia è una personalizzazione della libreria Bootstrap 5.2.3 che tiene conto delle indicazioni in termini di design, usabilità e accessibilità definite nelle nuove “Linee Guida per il design degli applicativi Web di Regione Lombardia di design per i siti internet e i servizi digitali della Pubblica Amministrazione”.
