Progetti

Showcase

Di seguito sono riportati alcuni progetti che sono risultato delle esperienze d'uso con le più recenti indicazioni di design e con Bootstrap Italia.

Sito di presentazione della app IO

Un sito realizzato in poche ore, anche grazie all'utilizzo di Bootstrap Italia, che presenta il progetto per la realizzazione dell'app del cittadino IO.

Prototipo del Comune di Cagliari

Un prototipo per il nuovo sito del Comune di Cagliari, costruito per fare ricerca e test con gli utenti, di cui è disponibile anche il codice sorgente in formato scaricabile.

Docs Italiawork in progress

Docs Italia, il progetto per la pubblicazione e la consultazione dei documenti pubblici digitali, è stato costruito su una versione preliminare di Bootstrap Italia.

Temi

Di seguito sono riportati alcuni progetti che semplificano la creazione di progetti con Bootstrap Italia e diversi CMS o framework per il web. Sono tutti in piena evoluzione e necessitano dell'aiuto di tutti per essere completati, resi pienamente conformi ed esenti da problemi. Se vuoi contruibire contattaci o crea una Pull Request sul repository!
