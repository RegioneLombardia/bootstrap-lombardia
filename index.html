Bootstrap Regione Lombardia

Versione di Bootstrap Italia per la Regione Lombardia per lo sviluppo di applicazioni web per la PA
  • v2.6.0 Download
  • GitHub

Bootstrap Regione Lombardia è una personalizzazione di Bootstrap Italia per la Regione Lombardia.

Il suo scopo è facilitare il design seriale del front-end degli applicativi della Regione Lombardia, utilizzando componenti conformi al Sistema di Identità Visiva Digitale di Regione Lombardia a cui ci si deve attenere per mantenere la coerenza e la continuità dell’ecosistema digitale dell’Ente.
